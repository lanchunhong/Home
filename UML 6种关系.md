# UML的6种关系

|                                         |
| --------------------------------------- |
| 泛化-子类父类-实三角                    |
| 实现-类和接口-虚三角                    |
| 组合-整体部分-实菱形-强拥有             |
| 聚合-整体部分-虚菱形-弱拥有             |
| 关联-拥有-实箭头                        |
| 依赖-使用-虚箭头                        |
| 泛化 = 实现 > 组合 > 聚合 > 关联 > 依赖 |
